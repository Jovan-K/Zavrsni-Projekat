Шта је веб програмирање?

Веб програмирање подразумева креирање веб страница уз помоћ програмских језика попут HTML-а, CSS-а и JavaScript-а.
На овом сајту ћете научити основе ових програмских језика уз примере који ће омогућити пробу истих.
На крају ћете бити способни да креирате просте веб странице и занимљиве веб апликације налик оних које можете видети на мом GitHub профилу.
Чар front-end веб програмирања лежи у томе да је ваш рад видљив визуелно и може да се обликује и стилизује по вашој жељи тако да подстиче креативност и машту.
HTML (HyperText Markup Language) се користи за описивање садржаја на веб страници, док CSS (Cascading Style Sheets) омогућава уређивање изгледа странице.
JavaScript је језик програмирања који се користи за стварање интерактивности на веб страницама.

Шта је HTML?

HTML је скраћено за HyperText Markup Language.
HTML је стандардни језик за креирање веб страница.
HTML описује структуру веб странице.
HTML се састоји из више елемената.
HTML преноси претраживачу како да прикаже садржај.

Објашњење примера:

Прва линија дефинише да је овај документ HTML5 документ.
head елемент садржи информације о HTML страници.
title елемент одредђује наслов HTML странице.
body је тело HTML странице и садржи сав видљив садржај.
h1 таг дефинише велики наслов.
p таг дефинише параграф.

Шта је CSS?

CSS је језик који користимо да стилизујемо веб страницу.
CSS је скраћено за Cascading Style Sheets.
CSS описује како HTML елементи треба да буду приказани на екрану, папиру или осталим медијумима.
CSS нам скраћује рад, може да контролише више веб страница истовремено.
CSS може да се смешта у спољашњим CSS фајловима.

Објашњење примера:

У стилу за body елемент боју позадине постављамо на светло-плаву.
background-color параметар дефинише боју позадине.
h1 наслов има бели текст и текст је центриран.
color параметар дефинише боју текста, а text-align: center центрира текст.
p параграф је фонта вердана и величине 20px.
font-family дефинише фонт текста, док font-size величину фонта.

Шта је JavaScript?

JavaScript је најпопуларнији програмски језк на свету.
JavaScript је програмски језик веб-а.
JavaScript је лак да се научи.

Објашњење примера:

На левој слици се налази делић HTML-а овог сајта.
Наслову је дат ID како би га селектор могао пронаћи, а дугмету позив функције коју можете видети у претходној слици.
Фунција мења .innerHTML особину селектованог наслова што буквално значи мењање садржаја између тагова за наслов у овом случају.
Можете видети баш како ова функција ради кликом на дугме на средини слике.
Функција је мало унапређена од оне са слике тако да поновним кликом на дугме враћамо претходни текст.
