CSS селектори:

CSS селектори...

Се користе да пронађу или "селектују" HTML елементе које желимо да стилизујемо.

Можемо их поделити у 5 категорије:

  • Прости слекетори (селектују елементе у зависности од имена, ид, или класе)

  • Комбинатор селектори (селектују елементе у зависности од посебног односа између њих)

  • Псеудо клас селектори (селектују елементе у зависности од стања елемента)

  • Псеудо елемент селектори (селектују и стилизују део елемента)

  • Атрибутни селектори (селектују елементе у зависности од атрибута или вредности атрибута)

Ова страница ће објаснити најпростије CSS селекторе.

Елементни, класни, ид и универзални селектор

Елементни селектор селектује све елементе тог типа.

Класни селектор селектује све елементе те класе HTML(class="klasa")

ИД селектор селектује елемент са тим ИД-ем HTML(id="id")

Универзални селектор селектује све елементе.
