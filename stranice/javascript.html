Кад корстити коју променљиву:

  • Увек декларишемо JavaScript променљиве са var, let и const.

  • Кључна реч var се користила у свом JavaScript коду од 1995 до 2015.

  • Кључне речи let и const су додате JavaScript-у 2015.

  • Ако желите да ваш код ради у старијим браузерима користите кључну реч var.

JavaScript функције:

Шта је JavaScript функција?

Функција је блок кода дизајниран да уради одређени задатак.

Функција се извршава када је нешто позове.

Синтакса функција:

JavaScript функција се дефинише кључном речју function, именом и на крају заградама ( ).

Заграде могу да садрже параметре одвојене запетом.

Код који се извршава се поставља у витичастим заградама: { }

Позивање функција:

JavaScript функција се извршава када нешто позове функцију:

  • Када се неки догађај изврши (корисник притисне дугме)

  • Када се позове из JavaScript кода.

  • Аутоматски (самопозвана)

Враћање вредности:

Када JavaScript достигне return кључну реч функција се прекида.

Ако је функција позвана као додела вредности некој променљиви вратиће вредност која стоји иза return кључне речи.
